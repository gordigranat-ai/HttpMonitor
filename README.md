# HTTP Monitor System

Простое приложение для мониторинга HTTP трафика с встроенным сервером и клиентом.

## Возможности

### HTTP Сервер
- Запуск HTTP сервера на любом свободном порту
- Обработка GET и POST запросов
- Автоматический поиск свободного порта, если указанный занят
- Статистика запросов (GET/POST, среднее время ответа)
- График нагрузки (запросов в минуту)
- Фильтрация логов по методу и коду ответа
- Экспорт логов в файл
- Автоматическое сохранение всех логов в `logs.txt`

### HTTP Клиент
- Отправка GET и POST запросов
- Поддержка JSON тела запроса
- Форматирование JSON ответов
- Готовые примеры для тестирования:
  - Посты JSONPlaceholder
  - Факты о котах (Cat Facts)
  - Отправка на свой локальный сервер

## Установка и запуск

### Требования
- Windows 7 и выше
- .NET 8.0 Runtime

### Запуск
1. Скачайте `HttpMonitor.exe`
2. Запустите файл (может потребоваться разрешение брандмауэра)
3. При необходимости запустите от имени администратора

## Как пользоваться

### Запуск сервера
1. Перейдите на вкладку **"HTTP Сервер"**
2. Введите номер порта (по умолчанию 8080)
3. Нажмите **"Запустить"**
4. Если порт занят, программа предложит использовать другой

### Отправка запросов
1. Перейдите на вкладку **"HTTP Клиент"**
2. Введите URL
3. Выберите метод (GET/POST)
4. Для POST заполните тело запроса (JSON)
5. Нажмите **"Отправить запрос"**

### Примеры запросов

| Кнопка | URL | Метод |
|--------|-----|-------|
| GET: Посты JSONPlaceholder | https://jsonplaceholder.typicode.com/posts | GET |
| GET: Один пост | https://jsonplaceholder.typicode.com/posts/1 | GET |
| GET: Cat Facts | https://catfact.ninja/fact | GET |
| POST: Отправить на свой сервер | http://localhost:порт/ | POST |

### Фильтрация логов
- **Фильтр по методу:** Все / GET / POST
- **Фильтр по коду ответа:** введите код (200, 404, 500 и т.д.)
- **Очистить:** сбросить фильтры

### Экспорт логов
Нажмите **"Экспорт"** для сохранения всех логов в файл:
logs_export_20260408_142530.txt

## Файлы программы

| Файл | Описание |
|------|----------|
| `logs.txt` | Автоматически сохраняемые логи |
| `logs_export_*.txt` | Экспортированные логи |
